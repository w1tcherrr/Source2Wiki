---
title: Asset Structure
---
[^1]This list contains all the asset types used by the game.

| Type                    | .Extension      | S1 Equivalent | Info                                                                                                                                                                                                                                                         |
| ----------------------- | --------------- | ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| VPK                     | vpk             | vpk           | Valve Pack. Is used to store content. One VPK file is limited to 2GB.                                                                                                                                                                                        |
| KV3                     | kv3             | kv3           | KeyValues3. Main file used for almost all asset types. Supports binary encodings, versioning and data annotation.                                                                                                                                            |
| VData Asset             | vdata           | txt           | VData asset. Form of Generic Data loaded by game for various stuff like weapons, heroes, abilities, decalgroups, styles, asset tags, precipitation, modifiers, entity subclasses, tagged sounds. Depending on the generic_data_type that comes from schemas. |
| Animation File          | vanim           | ani           | Used by Legacy VMDL. Unused.                                                                                                                                                                                                                                 |
| Animation Group         | vagrp           | ani           | Used by Legacy VMDL. Unused.                                                                                                                                                                                                                                 |
| Sequence Group          | vseq            | ani           | Used by Legacy VMDL. Unused.                                                                                                                                                                                                                                 |
| Particle Asset          | vpcf            | pcf           | Particles. Created in the Particle Editor.                                                                                                                                                                                                                   |
| Material Asset          | vmat            | vmt           | Materials. Created in the Material Editor.                                                                                                                                                                                                                   |
| Mesh Asset              | vmesh           | vvd           | Used by Legacy VMDL. Unused.                                                                                                                                                                                                                                 |
| Texture Asset           | vtex            | vtf           | Textures. Created by either the Material Editor, manual .vtex file or anything involving Panorama (VPDI/VDATA).                                                                                                                                              |
| Model Asset             | vmdl            | mdl           | Model. Created in the ModelDoc Editor. Compiled file embeds multiple blocks in itself.                                                                                                                                                                       |
| Physics Collision       | vphys           | phy           | Legacy Physics Collision file, used for Legacy VMDL and pre-Jun 6 2023 map compile. Unused.                                                                                                                                                                  |
| Sound Asset             | vsnd            | wav           | Sound file. Compiled out of a mp3/wav file. Can embed encoding data from `encoding.txt` and phoneme/morph data from `soundname.txt`.                                                                                                                         |
| Sound Container         | vsnd            | txt           | Soundcontainer. Shares the .vsnd extension with Sound, but is actually a .vdata file with a data type `CVoiceContainerBase.` A special kind of sound with rules/unique properties.                                                                           |
| Morphset Asset          | vmorf           | ?             | Used by Legacy VMDL. Unused.                                                                                                                                                                                                                                 |
| Resource Manifest       | vrman           | txt           | Resource Manifest. Tells the game to load specified assets when being read.                                                                                                                                                                                  |
| World Asset             | vwrld           | a bsp lump?   | World Asset. Part of a compiled map file. Loads defined in it worldnodes, lightmaps and entity lumps.                                                                                                                                                        |
| Worldnode Asset         | vwnod           | a bsp lump?   | World Node Asset. Part of a compiled map file. Loads defined worldnode models/scene objects.                                                                                                                                                                 |
| Worldvis Asset          | vvis            | a bsp lump?   | World Visibility Asset. Part of a compiled map file. Loads map visibility data. Visibility data are stored in voxel clusters.                                                                                                                                |
| Entitylump Asset        | vents           | a bsp lump?   | Entitylump Asset. Part of a compiled map file. Loads all defined map entities. Regular map entities are stored in `default_ents.vents` file, while point_template/world_layer entity lumps are stored in `hammeridofapointtemplate#entitylumpname.vents_c`.  |
| Surfaceproperties Asset | vsurf           | txt           | Surfaceproperties Asset. Has all the defined surface properties that Material Editor can use for materials.                                                                                                                                                  |
| Soundevent Script       | vsndevts        | txt           | Soundevent Script. Before being played in the game, your sound must be defined here. Then, that new soundevent can be defined anywhere else where it's needed.                                                                                               |
| Soundstack Script       | vsndstck        | txt           | Soundstack Script. Rules,operators and parameters for all the soundevents of the game.                                                                                                                                                                       |
| Mixgraph                | vmix            | ?             | Mixgraph Data. Created in the VMix Tool.                                                                                                                                                                                                                     |
| Postprocessing Settings | vpost           | ?             | Postprocessing settings. Created in the Postprocessing Editor. Controls postprocessing.                                                                                                                                                                      |
| Resource Remap Table    | vrmap           | ?             | Resource Remap Table. Unknown. Unused.                                                                                                                                                                                                                       |
| Panorama Style          | vcss            | ?             | Panorama Style. Compiled `.css` file, defines how a Panorama UI element will look.                                                                                                                                                                           |
| Panorama Layout         | vxml            | res           | Panorama Layout. Compiled `.xml` file, a skeleton of a Panorama UI element.                                                                                                                                                                                  |
| Panorama Dynamic Images | vpdi            | ?             | Panorama Dynamic Images. Acts the same purpose as Resource Manifest, but for Panorama images. They will be converted from source image asset to `.vtex_c` upon compile.                                                                                      |
| Javascript              | vjs             | ?             | Javascript. Created from a `.js` file.                                                                                                                                                                                                                       |
| Typescript              | vts             | ?             | Typescript. Created from a `.ts` file.                                                                                                                                                                                                                       |
| PulseGraphDef           | vpulse          | ?             | Pulse Graph Definition. Created in the Pulse Editor.                                                                                                                                                                                                         |
| Vector Graphic          | vsvg            | ?             | Vector Graphic. Created from a `.svg` file.                                                                                                                                                                                                                  |
| Particle Snapshot       | vsnap           | ?             | Particle Snapshot. Made in the Particle Editor, child asset for the Particle Asset, acts as a reference for the particle.                                                                                                                                    |
| ModelDoc Template       | vmodel_template | ?             | ModelDoc Template.                                                                                                                                                                                                                                           |
| ModelDoc Prefab         | vmdl_prefab     | ?             | ModelDoc Prefab, is essentially a `.vmdl`. Used by `.vmdl` files to share nodes.                                                                                                                                                                             |
| Animgraph Asset         | vanmgrph        | ?             | Animgraph1 Animgraph. Requires a `.vmdl` to compile. Animgraph 1 Graph, used for animation behavior, states, and etc. Currently being replaced by Animgraph2.                                                                                                |
| Animgraph Subgraph      | vsubgrph        | ?             | Animgraph1 Subgraph. Used by `.vanmgrph` files to share nodes, tags and parameters, is essentially a Animgraph group in a separate file.                                                                                                                     |
| Map Asset               | vmap            | vmf           | Map Asset. Compiled version does nothing and only references all the necessary stuff to load the map, while the source version is a Keyvalues2 VMAP file.                                                                                                    |
| Subrect Asset           | rect            | ?             | Subrect Asset. Created in the Subrect Editor, used for hot-spotting in Hammer.                                                                                                                                                                               |
| Composite Material Kit  | vcompmat        | ?             | Composite Material Asset. Used by CS2 for weapon skins.                                                                                                                                                                                                      |
| Econ Item               | econitem        | ?             | Economy Item. Used by CS2 Item Editor.                                                                                                                                                                                                                       |
| Smart Prop              | vsmart          | ?             | SmartProp. A prop with a defined set of rules and parameters in the VData Editor. Allows for much more control over a prop.                                                                                                                                  |
| Response Rules          | vrr             | txt           | Response Rules. Plays voicelines in response to events in the game.                                                                                                                                                                                          |
| Choreo Scene File Data  | vcdlist         | scenes.image  | Choreo Scene File Data. A combination of `.vcd` files for Choreo Scenes.                                                                                                                                                                                     |
| VCD File (Choreo)       | vcd             | vcd           | Valve Choreography Data. Stores choreo data for use with Faceposer.                                                                                                                                                                                          |
| Animgraph2 Skeleton     | vnmskel         | ?             | Animgraph2 Skeleton. Used by Animgraph2 for skeleton(s) and bonemask data.                                                                                                                                                                                   |
| Animgraph2 Clip         | vnmclip         | ?             | Animgraph2 Clip. Animation file used by Animgraph2. Can contain events to respond from graphs.                                                                                                                                                               |
| Animgraph2 Graph        | vnmgraph        | ?             | Animgraph2 Graph. Animgraph2's graph. Contains all the necessary behavior. Graph variations have a `vnmgraph.+variationname` suffix.                                                                                                                         |
| Animgraph2 Variation    | vnmvar          | ?             | Animgraph2 Variation. Deprecated.                                                                                                                                                                                                                            |
| VModuleMetadata         | vmmd            | ?             | Module Metadata. Contains all of the metadata about the module (like `server.dll`, `resourcecompiler.dll`) like resourcemanifests to load on startup and Pulse bindings.                                                                                     |
| CompiledShader          | vcs             | vcs           | CompiledShader. Contains all the shader data. Comes in multiple sets depending on the shader type. Stored in `shaders_platform/api.vpk`.                                                                                                                     |
| FlexSceneFile           | vfe             | vfe           | FlexSceneFile. Has expressions/phonemes data used by game/Faceposer.                                                                                                                                                                                         |
| ClosedCaptions          | dat             | dat           | Compiled Closed Captions file.                                                                                                                                                                                                                               |
| DAC Game Defs Data      | vdacdefs        | ?             | Dota Underlords Game Definitions file. Compiled version is a KV3 file.                                                                                                                                                                                       |
| Dota Visual Novels      | vdvn            | ?             | Visual Novels for various Dota 2 events.                                                                                                                                                                                                                     |
| Dota Patch Notes        | vdpn            | ?             | Patch Notes for Dota 2.                                                                                                                                                                                                                                      |
| Dota Hero List          | herolist        | ?             | Hero List for Dota 2.                                                                                                                                                                                                                                        |
| Artifact Item           | item            | ?             | Item for Artifact. Piece of items_game.txt.                                                                                                                                                                                                                  |
| Tools Asset Info        | bin             | ?             | Information file about all referenced assets used in Tools.                                                                                                                                                                                                  |
| Bitmap Font             | vfont           | vfont         | Bitmap font.                                                                                                                                                                                                                                                 |
| Processing Graph        | vpram           | ?             | Processing Graph. Precursor to legacy VMDL nodes.                                                                                                                                                                                                            |
| Surface Graph           | vsgraph         | ?             | Surface Graph. Unknown.                                                                                                                                                                                                                                      |

[^1]: **Information on some assets taken from [ValveResourceFormat](https://github.com/ValveResourceFormat/ValveResourceFormat)**
